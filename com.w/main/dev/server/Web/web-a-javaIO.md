# web-javaIO

## 分类

基于字节：InputStream&OutputStream

基于字符：Writer&Reader

基于磁盘：File

基于网络：Socket

## 磁盘IO方式

### 标准访问

内核空间到用户空间的拷贝

### 直接IO

Application直接操作磁盘，减少OS的拷贝

### 同步访问文件方式

读写都同步，性能差，当需要极高的数据安全性的时候采用

当写入成功返回OK

### 异步IO

提高应用程序效率，但磁盘访问效率并无变化

### 内存映射

将内存中的一块区域和磁盘文件管理起来，减少内核到用户空间的拷贝
