# Docker Overview

## 认识

Docker基础Linux容器技术LXC

复用操作系统，并提供隔离容器的作用

传统方式是在硬件上层进行进一步封装

## 特点

速度：启动速度快

资源利用：一台主机上同时运行数千个Docker

交付&部署：生产与运行环境的一致

虚拟化：不需要额外hypervisor支持，内核级虚拟化，拥有更高的性能和效率

迁移&扩展：各种平台的迁移

管理：少量的修改，增量的更新

## 基本概念

镜像 | 容器 | 仓库

镜像---只读

容器---镜像上层创建可写层

仓库---公有&私有

## DockerFile

DockerFile中每一条指令创建镜像的一层，不可超过127层

#---注释
